📌 CÓMO CONFIGURAR LA APP
1. Inicia sesión con tu usuario y contraseña
2. Configura tu Google Sheet (instrucciones abajo)
3. Registra tu salario y frecuencia de pago
4. Registra transacciones día a día
5. Revisa el dashboard para ver tu progreso mensual

La tasa BCV se actualiza automáticamente cada día usando la API pública de ExchangeRate.
📋 CONFIGURAR GOOGLE SHEETS
1. Ve a script.google.com y crea un nuevo proyecto
2. Pega el código del archivo google-apps-script.js incluido
3. Publica como Web App (ejecutar como: Yo, acceso: Cualquiera)
4. Copia la URL del Web App
5. En la app ve a ⚙️ Configuración y pega la URL
6. Haz clic en Probar conexión
